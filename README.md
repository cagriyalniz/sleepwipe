## About
sleepwipe is an open-source package that prevents the computer from going to sleep on macOS.

## Build

    git clone https://github.com/fleizean/sleepwipe.git
    cd sleepwipe
    export PATH=$PATH:/currentarea/sleepwipe/
    sleepwipe -help
